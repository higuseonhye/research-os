# Paper 1 — research question (public lock)

> **Status:** v0.9 · **Scope:** 6 months · Stage 1 measurement only

---

## Central question

> **When a task-relevant mismatch occurs, which response should start when to maximize the chance of successful resolution?**

---

## Measured object

**Intervention-conditioned recoverability** \(R_a(s,t)\) — empirical profiles from same-state counterfactual branches, not taxonomy label match alone.

---

## Method (Stage 1)

1. Shared pre-onset trajectory → task-relevant mismatch (target shift)
2. Fair fork at shared onset state
3. Branch responses (e.g. CONTINUE vs REPLAN) · optional delay axis
4. Terminal outcome vs **shifted** task target

Platform: Isaac Sim · ORBIT Dual-STAR Reach · scripted 6D IK-Rel · n=5/cell (smoke scale).
