## L1 RQ (one sentence)

**EN:** In non-average embodied settings, does intelligence emerge through a **failure lifecycle**—agentic exploration → generative dreaming → active validation → knowledge crystallization—and which stages can be **separated, measured, and ablated** without collapsing the pipeline?

**KR:** 비평균 embodied 환경에서 지능은 **에이전틱 탐색 → 생성적 dreaming → 능동적 검증 → 지식 결정화** lifecycle을 통해 형성되는가, 그리고 각 단계는 **분리·측정·ablation** 가능한가?

## Sub-RQ tree (L2)

### Stage 1 — Information generation
- **RQ 1.1 (agentic):** Can an agent design failure-eliciting curricula without human scenario lists?
- **RQ 1.2 (dreaming):** Does diffusion dreaming increase **informative failure yield** vs Gaussian (ReSYNC baseline)?

### Stage 2 — Knowledge crystallization
- **RQ 2.1:** Do failure–recovery trajectories refine PDDL preconditions (ReSYNC voice)?
- **RQ 2.2:** Can neural predicates be exported to interpretable rules (ILP)?

### Stage 3 — Validation & execution
- **RQ 3.1 (interactive perception):** Does physics-changing perception beat viewpoint-only @ fixed **S**?
- **RQ 3.2 (001):** Which intervention menu maximizes successful resolution @ **S**?
